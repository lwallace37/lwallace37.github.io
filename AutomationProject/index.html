Automation Project Integration of a SCARA Robot

Project Description

While at Metallon Inc. I took lead on a project to increase the efficeincy of assembing parts by integrating an Epson SCARA 3-Axis robot into the system.

Project Details

  • Reduced robot cycle time by 30%
  • Increased daily production by 50%
  • Programmed the motion of the Epson SCARA Robotic Arm
  • Programmed the Productivity2000 PLC
  • Programmed Cognix VisionView Part checking camera to check for deformities
